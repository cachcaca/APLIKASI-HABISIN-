# APLIKASI-HABISIN-
**Habisin!**
Habisin! adalah aplikasi mobile yang dirancang untuk membantu pengguna mengurangi food waste dengan mengelola stok makanan yang dimiliki dirumah.
Aplikasi ini memungkinkan pengguna mencatat persediaan bahan makanan, memantau tanggal kedaluwarsa, serta menerima pengingat agar makanan dapat digunnakan sebelum dibuang.

**Fitur utama:**
- Manajemen stok bahan makanan
- Pengingat tanggal kedaluwarsa
- Pencarian dan pengelompokkan bahan makanan
- Monitoring persediaan makanan
